# Unity 字典

一個以 YouTube 短片為核心的 Unity 功能查詢工具。每個條目包含一支教學短片與簡短說明，讓你快速掌握 Unity 的各種功能與概念，不需要看完整堂課。


## 線上瀏覽

https://hkhorazon.github.io/UnityDictionary/

## 功能

- **快速瀏覽**：每個條目只有一支短片 + 數句說明，節省時間
- **分類篩選**：依功能類別（Lighting、Physics、UI 等）過濾條目
- **關鍵字搜尋**：即時搜尋標題、說明與標籤
