# 115年度 教學輔助媒材製作獎補助 - 申請書內容草案 (新案修正版 v2)

## 基本資料
- **計畫名稱**：遊戲引擎入門圖解：Unity 十大核心基礎極短篇
- **申請類別**：數位科技教學輔助媒材 (影音串流)
- **適用課程**：遊戲引擎基礎概論 (Introduction to Game Engines)
- **預計應用學期**：115學年度第1學期

## 一、教學目標 (Teaching Objectives)
1. **降低入門門檻**：針對完全零基礎的初學者，將 Unity 介面中「最常用」但「最容易混淆」的基礎操作，濃縮為直觀的短影音，消除對複雜軟體的恐懼感。
2. **視覺化操作指引**：透過影片示範「滑鼠游標位置」與「快捷鍵操作」，解決紙本講義或靜態投影片無法呈現動態操作細節的問題。
3. **建立標準化流程**：為課程建立一套標準的「課前預習」媒材，確保每位學生在進入實作課前，都具備相同的基礎視窗操作能力。

## 二、製作內容與特色 (Content & Features)

### 1. 影片規劃 (精選 10 支基礎短片)
本教材將製作 10 支時長約 3-5 分鐘的教學短片，聚焦於「第一次打開 Unity 必須知道的事」：

| 編號 | 影片主題 | 學習重點 |
| :--- | :--- | :--- |
| 01 | **認識你的工作檯** | Scene / Game / Hierarchy / Inspector 四大視窗功能導覽 |
| 02 | **上帝視角的操作** | 熟練 Scene 視窗運鏡：旋轉 (Alt+左鍵)、平移 (中鍵)、縮放 (滾輪) |
| 03 | **物體的移動魔法** | QWERT 工具列詳解：移動 (Move)、旋轉 (Rotate)、縮放 (Scale) 的操作手感 |
| 04 | **無中生有：建立物件** | 如何在場景中建立 Cube、Sphere 等基本幾何體 (Primitive Objects) |
| 05 | **物件的親子關係** | 理解 Hierarchy 視窗中的 Parent-Child 階層結構與座標影響 |
| 06 | **幫物件穿衣服** | 材質球 (Material) 的建立、顏色修改與拖曳套用 |
| 07 | **讓積木掉下來** | 剛體 (Rigidbody) 元件介紹：重力 (Gravity) 開關的影響 |
| 08 | **穿透與阻擋** | 碰撞器 (Collider) 選項：Is Trigger 的勾選與差異 |
| 09 | **整理你的房間** | Project 視窗管理：建立資料夾與 Asset Store 資源匯入基礎 |
| 10 | **保存你的世界** | Save Scene (場景存檔) 與 Save Project (專案存檔) 的區別與重要性 |

### 2. 教學特色
- **新手友善 (Beginner Friendly)**：屏除所有程式碼 (Coding)，專注於介面操作與物理屬性設定，適合一年級新生。
- **畫中畫示範 (Picture-in-Picture)**：使用高畫質攝影機同步錄製鍵盤與滑鼠的操作手勢，與螢幕畫面同步呈現，幫助學生直觀學習。
- **高音質解說**：使用專業級電容式麥克風錄製旁白，提供清晰的語音引導。

## 三、預期成效 (Expected Benefits)
1. **提升課堂流暢度**：學生透過預習短片先熟悉介面操作，教師在課堂上無須重複教導「如何旋轉視角」等機械式操作。
2. **具體產出**：
   - 10 支高畫質 (4K/60fps) Unity 基礎教學短片。
   - 建立專屬 YouTube 播放清單，供學生永久複習。
3. **質化效益**：建立學生對遊戲引擎的自信心，從「害怕按錯按鈕」轉變為「敢於嘗試操作」。
